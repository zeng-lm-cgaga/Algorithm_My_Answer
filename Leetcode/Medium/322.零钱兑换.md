# Leetcode ：322.零钱兑换

## 一、题目链接
# [LeetCode 322.零钱兑换](https://leetcode.cn/problems/coin-change/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
找到能凑成要求值的最少硬币个数

## 三、解题思路与代码实现
### 1. 核心思路
- 完全背包：
    先遍历物品再遍历背包，使用一维dp数组
    - 状态表示：当前金额的最少硬币个数
    - 状态初始化：当为0元时硬币个数为0即dp[0] = 0。
    - 状态转换：有当前位置和当前背包大小减去物品值的位置的个数的最小值转换而来即：
        dp[j] = min(dp[j], dp[j-coins[i] + 1]);

### 2. 代码实现
#### 完全背包：
```C++
class Solution {
public:
    int coinChange(vector<int>& coins, int amount) {
        vector<int> dp(amount + 7, INT_MAX);
        dp[0] = 0;
        for( int i = 0; i < coins.size(); ++i) {
            for( int j = coins[i]; j <= amount; ++j ) {
                if( dp[j-coins[i]] < INT_MAX)
                    dp[j] = min( dp[j], dp[j-coins[i]] + 1);
            }
        }
        return dp[amount] == INT_MAX ? -1 : dp[amount];
    }
};
```
- 时间复杂度：O(mn)
- 空间复杂度：O(n)
