# Leetcode ：347.前K个高频元素

## 一、题目链接
# [LeetCode 347.前K个高频元素](https://leetcode.cn/problems/top-k-frequent-elements/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
将数组排序

## 三、解题思路与代码实现
### 1. 核心思路
- 堆排序（优先队列）：
    将各个字数出现的次数使用哈希表存储（数字，个数），然后定义一个优先队列还有自定义仿函数作为比较器，使优先队列以个数为标准建立小根堆，然后只需要将优先队列的值控制在K个，就是答案

### 2. 代码实现
#### 堆排序（优先队列）：
```C++
class Solution {
public:
    class Mycomparion {
    public:
        bool operator()( const pair<int,int> &a , const pair<int,int> &b ) {
            return a.second > b.second;
        }
    };
            
    vector<int> topKFrequent(vector<int>& nums, int k) {
        unordered_map<int,int> map;
        for( auto i : nums ) 
            map[i] ++;

        priority_queue<pair<int,int> , vector<pair<int,int>> , Mycomparion> pri_que;

        for( auto it = map.begin() ; it != map.end() ; ++it ) {
            pri_que.push(*it);
            if( pri_que.size() > k )
                pri_que.pop();
        }

        vector<int> result(k);

        for( int i = k - 1 ; i >= 0 ; --i  ) {
            result[i] = pri_que.top().first ;
            pri_que.pop(); 
        } 
        return result;
    }
};
```
- 时间复杂度：O(nlogk)
- 空间复杂度：O(n)
