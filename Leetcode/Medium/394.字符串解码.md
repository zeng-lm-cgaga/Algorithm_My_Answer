# Leetcode ：394.字符串解码

## 一、题目链接
# [LeetCode 394.字符串解码](https://leetcode.cn/problems/decode-string/description/)

## 二、题目描述
根据输入的字符串进行编解码

## 三、解题思路与代码实现
### 1. 核心思路
- 双栈法：
    1. 定义一个存放倍数的栈，一个存放字符的栈还有
    2. 当遇到'[' 时，将当前的子串和数字分别压入字符栈和数字栈中
    3. 当遇到']' 时，将两个栈顶的值弹出，并且将弹出的值加上倍数与当前子串
    4. 遇到数字或者字母就加到当前倍数和当前子串

### 2. 代码实现
#### 双栈法：
```C++
class Solution {
public:
    string decodeString(string s) {
        stack<int> count;
        stack<string> str;
        string currentString = "";
        int k = 0;
        for( char c : s) {
            if( isdigit(c)) {
                k = k * 10 + (c - '0');
            }
            else if( c == '[' ) {
                count.push(k);
                str.push(currentString);
                currentString = "";
                k = 0;
            }
            else if( c == ']' ) {
                int repeatTime = count.top();
                count.pop();
                string temp = str.top();
                str.pop();
                for(int i = 0; i < repeatTime; i++ ) {
                    temp += currentString;
                }
                currentString = temp;
            }
            else {
                currentString += c;
            }
        }
        return currentString;
    }
};
```
- 时间复杂度：O(n)
- 空间复杂度：O(n)
