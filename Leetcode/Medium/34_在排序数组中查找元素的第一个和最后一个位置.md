# Leetcode ：34.在排序数组中查找元素的第一个和最后一个位置

## 一、题目链接
# [34.在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
在递增排列的整数数组中找目标值的区间


## 三、解题思路与代码实现
### 1. 核心思路
- 二分查找法
    1. 使用常规二分法寻找目标值
    2. 找到目标值时，使用双指针从目标值向两边扩散寻找区间

### 2. 代码实现
#### 二分算法：
```C++
class Solution {
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        if(nums.size() == 0 ) return {-1, -1};
        int left = 0, right = nums.size() - 1;
        while(left <= right) {
            int middle = (left + right) / 2;
            if( nums[middle] == target ) {
                int i = middle, j = middle;
                while( i >= 0 && nums[i] == target ) i--;
                while( j < nums.size() && nums[j] == target ) j++;
                return {i+1, j-1};
            }
            else if(nums[middle] > target) right = middle - 1;
            else left = middle + 1;
        }
        return {-1, -1};
    }
};
```
- 时间复杂度：O(log n)
- 空间复杂度：O(1)
