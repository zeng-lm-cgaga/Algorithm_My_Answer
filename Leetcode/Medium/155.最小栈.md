## 二、题目描述
实现一个最小栈类，多了一个函数接口可以获得此时栈中的最小值

## 三、解题思路与代码实现
### 1. 核心思路
- 实现一个基础栈的同时增加一个存储最小值的辅助栈，并且类初始化时先向辅助栈添加一个最大值如（INT_MAX），每次加入元素时，将元素加入基础栈，而且在辅助栈中加入元素和栈顶中的最小值，这样每次加入元素都会添加一个最小值在辅助栈，出栈基础栈和辅助栈一起出栈，这样就可以随时访问当前栈的最小值

### 2. 代码实现
#### 最小栈：
```C++
class MinStack {
    stack<int> min_stack;
    stack<int> st;
public:
    MinStack() {
        min_stack.push(INT_MAX);
    }
    
    void push(int val) {
        st.push(val);
        min_stack.push(min(min_stack.top(), val));
    }
    
    void pop() {
        st.pop();
        min_stack.pop();
    }
    
    int top() {
        return st.top();
    }
    
    int getMin() {
        return min_stack.top();
    }
};
```
