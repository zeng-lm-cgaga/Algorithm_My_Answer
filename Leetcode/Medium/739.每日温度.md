# Leetcode ：739.每日温度

## 一、题目链接
# [LeetCode 739.每日温度](https://leetcode.cn/problems/daily-temperatures/description/)

## 二、题目描述
求每一天到比自己温度大的那一天隔了多少天

## 三、解题思路与代码实现
### 1. 核心思路
- 单调栈:
    使用单调递减栈，栈中保存每一天的下标，只要栈顶的元素比当天的温度低，保存栈顶下标答案为当日下标减去栈顶下标，最后检查栈是否为空，将栈的所有下标设置为0表示后面没有比他大的温度


### 2. 代码实现
#### 快排算法：
```C++
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        stack<int> st;
        vector<int> result( temperatures.size() , 0 );
        st.push( 0 );
        for( int i = 1 ; i < temperatures.size() ; ++i ) {
            while( !st.empty() && temperatures[i] > temperatures[st.top()]   ) {
                result[st.top()] = i - st.top();
                st.pop();
            }
            st.push(i);
        }
        while( !st.empty() ) {
            temperatures[st.top()] = 0;
            st.pop();
        }
        return result;
    }
};
```
- 时间复杂度：O(n)
- 空间复杂度：O(n)
