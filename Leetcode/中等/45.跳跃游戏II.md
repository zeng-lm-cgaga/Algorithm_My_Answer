# Leetcode ：45.跳跃游戏

## 一、题目链接
# [LeetCode 45.跳跃游戏](https://leetcode.cn/problems/jump-game-ii/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
找到到达数组最后一个值的最小跳跃次数

## 三、解题思路与代码实现
### 1. 核心思路
- 贪心算法：
    先寻找当前能跳到的最大边界，然后再当前区间[i, curdistance]中找最大的下一步的边界，即nextdistance = max(nextdistance, nums[i] + i),
    且当当前位置i==curdistance时步数加1，更新边界为下一次跳跃的最大边界。当nextdistance大于等于重点时，步数即为最小值。

### 2. 代码实现
#### 贪心算法：
```C++
class Solution {
public:
    int jump(vector<int>& nums) {
        if( nums.size() == 1 ) return 0;
        int curdistance = 0;
        int nextdistance = 0;
        int ans = 0 ;
        for( int i = 0 ; i < nums.size() ; ++i ) {
            nextdistance = max( i + nums[i] , nextdistance ) ;
            if( i == curdistance ) {
                ans++;
                curdistance = nextdistance;
                if( nextdistance >= nums.size() - 1 ) break; 
            }
        }
        return ans;
    }
};
```
- 时间复杂度：O(n)
- 空间复杂度：O(1)
