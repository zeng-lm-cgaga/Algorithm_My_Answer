# Leetcode ：416.分割等和子集

## 一、题目链接
# [LeetCode 416.分割等和子集](https://leetcode.cn/problems/partition-equal-subset-sum/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
判断原数组是否能分成两个总和相等的子数组

## 三、解题思路与代码实现
### 1. 核心思路
- 01背包：
    - 状态表示：当前背包容量下装的最大值
    - 状态初始化：都初始化为0
    - 状态转移：有当前状态和当前背包容量减去物品大小的状态的最大值转换而来
    - 将数组总和除以2就是目标值，查看dp数组中目标值的状态，值是否等于目标值

### 2. 代码实现
#### 01背包：
```C++
cclass Solution {
public:
    bool canPartition(vector<int>& nums) {
        int sum = 0;
        vector<int> dp(10001, 0);
        for (int i = 0; i < nums.size(); i++) {
            sum += nums[i];
        }

        if (sum % 2 == 1) return false;
        int target = sum / 2;

        for(int i = 0; i < nums.size(); i++) {
            for(int j = target; j >= nums[i]; j--) {
                dp[j] = max(dp[j], dp[j - nums[i]] + nums[i]);
            }
        }
        return dp[target] == target;
    }
};
```
- 时间复杂度：O(mn)
- 空间复杂度：O(n)
