# Leetcode ：20.有效的括号

## 一、题目链接
# [20.有效的括号](https://leetcode.cn/problems/valid-parentheses/)

## 二、题目描述
检查括号字符串是否有效

## 三、解题思路与代码实现
### 1. 核心思路
- 使用栈当遇到左括号时压栈，遇到右括号将栈顶元素与右括号检查是否匹配

### 2. 代码实现
#### 栈：
```C++
class Solution {
public:
    bool isValid(string s) {
       stack<char> st;
       if( s.size() % 2 != 0 ) return 0;
       for( auto i : s ) {
            if( i == '{' || i == '[' || i == '(' )
                st.push(i);
            else {
                if( st.empty() ) return false ;
                char top = st.top() ;
                st.pop();
                if( (i == ')' && top != '(') || (i == ']' && top != '[') || (i == '}' && top != '{') )
                    return false ;                
            }
       }
       return st.empty() ? true : false ;
    }
};
```
- 时间复杂度：O(n)
- 空间复杂度：O(n)
