## 二、题目描述
寻找数组中的最小值

## 三、解题思路与代码实现
### 1. 核心思路
- 使用二分查找法

### 2. 代码实现
#### 二分查找：
```C++
class Solution {
public:
    int inventoryManagement(vector<int>& stock) {
        int left = 0, right = stock.size() - 1;
        while( left < right ) {
            int mid = (left + right) / 2;
            if( stock[right] > stock[mid] ) right = mid;
            else if( stock[right] < stock[mid] )left = mid + 1;
            else right--;
        } 
        return stock[left];
    }
};
```
- 时间复杂度：O(log n)
- 空间复杂度：O(1)
