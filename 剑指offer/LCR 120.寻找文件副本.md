## 二、题目描述
寻找数组中的最小值

## 三、解题思路与代码实现
### 1. 核心思路
- 原地交换方法，通过调整索引映射调整位置，使数组下标元素的值等于下标值，当遇到当前值作为索引元素的值等于当前值时，说明有重复值哦，直接返回

### 2. 代码实现
#### 原地交换：
```C++
class Solution {
public:
    int findRepeatDocument(vector<int>& documents) {
       int i = 0;
       while( i < documents.size() ) {
            if( documents[i] == i ) {
                ++i;
                continue;
            }
            else if( documents[documents[i]] == documents[i] )
                return documents[i];
            else
                swap( documents[documents[i]], documents[i]);
       }
       return -1;
    }
};
```
- 时间复杂度：O(N)
- 空间复杂度：O(1)
