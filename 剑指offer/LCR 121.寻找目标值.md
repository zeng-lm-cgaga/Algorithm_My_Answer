# Leetcode ：LCR 121.寻找目标值

## 一、题目链接
# [LCR 121.寻找目标值](https://leetcode.cn/problems/er-wei-shu-zu-zhong-de-cha-zhao-lcof/description/?envType=problem-list-v2&envId=GzFbGrmM)

## 二、题目描述
在二维数组中寻找到目标值

## 三、解题思路与代码实现
### 1. 核心思路
- 将二维数组逆时针转45°，当作类二叉搜索树，因为旋转后，每个节点都大于其左下边的值，小于右下边的值，将右上角的值作为根节点进行搜索

### 2. 代码实现
#### 技巧：
```C++
class Solution {
public:
    bool findTargetIn2DPlants(vector<vector<int>>& plants, int target) {
        if( plants.size() == 0 ) return false;
        int i = 0, j = plants[0].size() - 1;
        while( j >= 0 && i < plants.size()) {
            if( plants[i][j] > target ) 
                j--;
            else if( plants[i][j] < target ) 
                i++;
            else return true;
        }
        return false;
    }
};
```
- 时间复杂度：O(M + N)
- 空间复杂度：O(1)
