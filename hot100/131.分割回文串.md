# Leetcode ：131.分割回文串

## 一、题目链接
# [LeetCode 131.分割回文串](https://leetcode.cn/problems/palindrome-partitioning/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
找到字符串中所有的回文子串


## 三、解题思路与代码实现
### 1. 核心思路
使用回溯算，传参startindex遍历分割字符串的头，然后在函数体里遍历长度，符合条件的都加入答案中


### 2. 代码实现
#### 回溯算法：
```C++

class Solution {
private:
    vector<string> path;
    vector<vector<string>> result;
    void backtracking( string& s , int startindex ) {
        if( startindex >= s.size() ) {
            result.push_back(path);
            return;
        }
        for( int i = startindex ; i < s.size() ; ++i) {
            if( isPalindrome( s , startindex , i ) ) {
                string cur = s.substr( startindex , i - startindex + 1 );
                path.push_back( cur );
                backtracking( s , i + 1 );
            }
            else {
                continue;
            }
            path.pop_back();
        }
    } 
    bool isPalindrome( string& s , int start , int end ) {
        for( int i = start ,  j = end ; i < j ; ++i , --j ) 
            if( s[i] != s[j] ) return false;
        return true;
    }
public:
    vector<vector<string>> partition(string s) {
        path.clear();
        result.clear();
        backtracking( s , 0 );
        return result;
    }
};
```
