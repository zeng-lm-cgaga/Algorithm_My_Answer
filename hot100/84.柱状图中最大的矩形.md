# Leetcode ：84.柱状图中最大的矩形

## 一、题目链接
# [84.柱状图中最大的矩形](https://leetcode.cn/problems/largest-rectangle-in-histogram/description/)

## 二、题目描述
在高度不一宽度都为一的柱状图中找最大的矩形

## 三、解题思路与代码实现
### 1. 核心思路
- 单调栈：
    1. 使用递增单调栈，现在原数组左右两侧插入数字0，防止元素组全是递增元素导致逻辑处理复杂。
    2. 我们要找每个柱子能形成的最大矩形，需要找到每一个柱状体的左右边界，如果当前元素小于栈顶元素，就将栈顶元素取出，并计算宽度及面积，最后返回最大的矩形面积

### 2. 代码实现
#### 快排算法：
```C++
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        stack<int> st;
        int result = 0;
        heights.insert( heights.begin() , 0 );
        heights.push_back( 0 );
        st.push(0);
        for( int i = 1 ; i < heights.size()  ; ++i ) {
            while( heights[i] < heights[st.top()] ) {
                int mid = st.top();
                st.pop();
                int h = heights[mid] ;
                int w = i - st.top() - 1;
                result = max(result, w * h);
            }
            st.push(i);
        }
        return result;
    }
};
```
- 时间复杂度：O(n)
- 空间复杂度：O(n)
