# Leetcode ：35.搜索插入位置

## 一、题目链接
# [LeetCode 35.搜索插入位置](https://leetcode.cn/problems/search-insert-position/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
在数组中查找元素的下标，找到返回没找到返回插入的位置

## 三、解题思路与代码实现
### 1. 核心思路
二分查找：当找到的时候直接返回下标，若没有找到，则比较left下标对应的值与目标值大小比较，因为如果没有找到，两种情况：
1. nums[left] > target，那就直接插入left下标处
2. nums[left] < target，那就插入left+1下标处

### 2. 代码实现
#### 二分查找算法：
```C++

class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int left = 0 , right = nums.size() - 1;
        while( left < right ) {        
            int middle = (left + right) / 2;
            if( target > nums[middle] ) {
                left = middle + 1;
            }
            else if( target < nums[middle]) {
                right = middle - 1;
            }
            else return middle;
        }
        return nums[left] < target ? left + 1 : left;
    }
};
```
