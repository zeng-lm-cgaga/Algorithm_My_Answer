# Leetcode ：51.N皇上

## 一、题目链接
# [LeetCode 51.N皇上](https://leetcode.cn/problems/n-queens/description/?envType=study-plan-v2&envId=top-100-liked)

## 二、题目描述
找到能容下N个皇上的棋盘


## 三、解题思路与代码实现
### 1. 核心思路
因为每行只能放入一个，传参遍历行数，函数体里遍历列数，然后每次放入一个皇上时判断该皇上是否有效，进行下一步或者回溯


### 2. 代码实现
#### 回溯算法：
```C++

class Solution {
private:
    vector<vector<string>> result;
    void backtracking( int n , int row , vector<string>& chessboard) {
        if( row == n ) {
            result.push_back( chessboard );
            return;
        }
        for( int i = 0 ; i < n ; ++i) {
            if(valid( row , i , chessboard , n  )) {
                chessboard[row][i] = 'Q';
                backtracking( n , row + 1 , chessboard );
                chessboard[row][i] = '.';
            }
        }
        return ;
    }
    bool valid( int row , int col , const vector<string>& chessboard , int n ) {
        for( int i = 0 ; i < row ; ++i) {
            if( chessboard[i][col] == 'Q' ) return false;
        }
        for( int i = row - 1 , j = col - 1 ; i >= 0 && j >= 0 ; --i , --j ) {
            if( chessboard[i][j] == 'Q' ) return false;
        }
        for( int i = row - 1 , j = col + 1 ; i >= 0 && j < n ; --i , ++j ) {
             if( chessboard[i][j] == 'Q' ) return false;
        }
        return true;
    }
public:
    vector<vector<string>> solveNQueens(int n) {
        result.clear();
        vector<string> chessboard( n , string( n , '.'));
        backtracking( n , 0 , chessboard);
        return result;
    }
};
```
